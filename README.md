Project Name: Shift

Description: A "realisitc" simulation of car racing where the player can customize the supposed car in its stats such as torque, stability control, and braking effectiveness. The project is built under the Godot 4.2 engine and based off of https://github.com/DAShoe1/Godot-Easy-Vehicle-Physics

Features: 1 Car Model, 1 Track, A customizer menu.

How to run: The essentials needed are the Godot 4.2+ engine and the project files.
            1. you must put the project files in any file and name it "Shift".
            2. You must open the Godot engine and import the files.
            3. You must run the project and compile it any format that user wants and that is supported.
            4. Run the compiled files and test out the test simulation.
